Allows you to lock site display to users with ad Blockers installed and increase your earns.

== Description ==
NS AdBlock Blocker helps you to detect site visitors who have installed and activated adBlocking extensions.
You can require users to turn off their ad blocker before allowing them into the site.
Substantially this plugin, if adBlock is activated, redirect users into blocked page.

== Installation ==

1. Upload the entire 'ns-adblock-blocker' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Ad Block Blocker'
